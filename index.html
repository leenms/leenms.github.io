两个刷屏日（一）：支付宝 2017 年账单

2018-01-03

并没有什么固定的要刷朋友圈的习惯，今天是支付宝年度账单日，我知道得刷一刷了。因为如果没有朋友圈，从作为一项运营手段的角度来看，支付宝账单日的运营效果可以降低一半不止。

在十来页账单内容里，有两点是人们晒的最多的，一是账单相关的数字，二是所谓智能预测的 2018 年度关键词。前者，基本是给账单成绩还不错的人晒的，这里我想聊的是第二部分。

在所有年度关键词里，「能干」以最高的出镜率占领了我的朋友圈。值得一提的是，测出「能干」的人津津乐道，测出其他任何词语的人同样引以为趣。关键就在这一点，因为其实拿到什么词毫不重要，无论怎样，都可以根据你的交易数据匹配出一个的夸赞你的词。只是人们都在抓紧这样的机会晒自己而已。

社交网络解决的深层次需求就是如何更加高效地向外界展现「我想让他们看到的（更好的）我」。或者说，这个需求就是社交网络创造出来的。

所谓的 2018 年度关键词，准不准，没有意义，符合人们对于自己的预期（幻想），或者人们愿意展现给外界看到的，才有意义。每个人的词语也许都与另外 8462443 个人相同，也不重要，只要每人都有一个还不错、挺有趣的词语，就足够了。

[ 2018-01-27 补充 ] 旅行青蛙里那只蛙今天到底是会出去旅行，还是会削一下午的铅笔，旅行带回来的手信是辣椒还是草莓，其实全都一个样，但每个人都觉得自己的蛙是最令人操心的同时又是最高冷的，每个人都既盼着它出门又盼着它回家。因为这样一只蛙，让你感受到可以为它付出，也可以被它需要。而在这个过程中，你最大化地感受到了你自己的重要性、独特性。某种程度上来说，这只撑着一顶荷叶的蛙，以及那个年度关键词，何以戳中我们，其实是相似的。那就是每个人都希望自己是优秀的、重要的、独特的，激发出这一点，就成了，哪怕只是营造了一些错觉。

两个刷屏日（二）：网易云音乐

2018-01-03

原来昨天还被网易云音乐刷屏了，那个以精准的个性化推荐、和（诙）谐的社区氛围乃至复古黑胶唱片 UI 等等著称的网易云。

在我的印象中，同时还有两个词语代表着网易云音乐，那就是情怀、高端。但我发现，云村村民在各自的听歌喜好、品味、习惯上呈现出来的完全是另一派画风。有最爱本土民谣的，有最爱华语黄金年代的，有最爱时下欧美圈的，有最爱二次元的，有最爱钢琴曲的，同时也有最爱网络歌曲的。那景象，你说它是博览会也罢，是大商场也行。总之一派纷呈，与我的想象相距甚远。

网易云音乐的用户量已经经历了最快的增长期，正在慢慢走向它当前的天花板。一个多月前才刚刚宣布达到了 4 亿注册用户数，这是很厉害的成绩。而随着用户量的增长，用户结构呈现出丰富性是必然的，这也是成功的产品走向市场成熟阶段的必然。

而这种能在用户量上迅速达到的成功，很大一种程度是因为，不管什么风格的音乐，其能够承载的核心用户行为是完全一致的。什么歌不都是播出来用耳朵听嘛，而歌单推荐、评论等，这些需求也都是一致的，用户在这里都可以互相增强认同感。也就是说在一个完整的用户画像里，听什么流派的音乐只是其中一项属性，并且这一项属性与其他属性耦合度很低，因此从拓展不同流派听众的用户群体上来说，不需要额外的边际成本，甚至是可以自然发展的。因此今天在朋友圈里发现的情况，就不难理解了。不是因为用户听的歌有情怀，这个产品才有情怀，而是这个产品自身能够创造情怀。

所谓优秀的产品不一定是爆款，但各种人群都会拥簇的产品一定意味着它在当下聚焦的核心价值层面取得了成功。

最后一天

2017-12-31

没有例外，我也像大多数人一样把今天看作一个特别的日子。

其实这也是平常的一天，但我们对仪式感和新鲜感有着天生的追求，于是纷纷用「跨年」这场仪式祭奠过去迎接未来，仿佛钟声敲响那一刻之后就能摇身一变成为一个崭新的自己。所以一定要疯狂又体面、郑重又堂皇地，跟自己说一声再见，向未来问一声好。这个仪式似乎每个人都有，然而曾经的我很不以为意。因为每当这一天这一刻的来临，我便深知，其实还有另有一个我停留在原地。他不愿走，我又怎能走得了。

曾经那个恶魔的旧的自己，因为一些无法逃避的事情，冷漠沉郁、自厌自愤，坚信世间一切都是不自知的人们用傲慢、谎言编著的荒诞假象。

「我的精神分裂治好了，现在的我和我过得很好。」看到过这样一句话，带着一种风靡过空间说说或者个性签名的感觉，现在看来微妙又有点好笑。但我很坦诚的讲，的确很长时间以来，原本那个阴暗的我都依靠着另一个我花尽力气从世界汲取养分，然后白白献祭给他，以此达到一种可以维持正常心理状态的平衡。但现在的我，他正在努力变得像其他人一样，纯粹、热情、喜乐，向往美好并乐于与人们共同拥有这个繁华多彩的世界，而这一切的至珍至贵，他也不再会全盘交托给曾经那另一个阴暗的我。不值得。

整整一年前，我在朋友圈写了一条「2017 快乐」，当时赋予它们的「希望 2017 年过的简单快乐」的使命，尽管完成得一般，也总算可以了结了。然后回头告诫自己，2018 别再有什么非分之想了，没有免费的午餐，没有简单的快乐，一切都要靠自己亲身经历、亲手创造。

「看你飞远去，看你离我而去，原来你生来就属于天际。」《大鱼》在安静的房间里回响，周深的声音男声也不是女声也不是，听起来包裹着一层气息，可是非常辽阔深邃，有着令人动容的绵延之力。此刻我看见大鱼在天空的海洋里遨游盘旋不愿离开，一边挽留着时间，一边又有些焦急和不知所措。然而我知道，大鱼终会离开的。

如果在这一年的最后一天，必须要选择一种心态，曾经的我，会选择「这一年终于过去了」，而现在的我，会选择「新的一年又到啦」。
